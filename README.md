# Tauri + React + Typescript

This template should help get you started developing with Tauri, React and Typescript in Vite.

## Running the app

Install deps:

```bash
bun install
```

Run the web app (Vite dev server):

```bash
bun run dev
```

Run the desktop app (Tauri):

```bash
bun run tauri dev
```

Build production assets:

```bash
bun run build
```
